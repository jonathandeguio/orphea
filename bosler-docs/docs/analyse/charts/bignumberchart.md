# Big Number

## **Big Number Chart**
> A **Big Number Chart** is a graphical representation that displays a single, large numerical value to highlight important metrics or key performance indicators (KPIs). This chart type is useful for showing summary statistics or significant values at a glance. This documentation will guide you through the configuration options available for creating a big number chart in our application and provide instructions on how to create one effectively.

### **Operations available in charts**

> **Display Value:** The **Display Value** option allows you to choose the key metric or number that will be prominently shown in the big number chart. You can select from the following parameters:
>> **Metric**: Choose the specific numerical value to display, such as Total Revenue, Total Sales, or Average Score.
**Aggregation Method**: Provides methods to summarize the data, including:
        - Count: Total count of rows.
        - Distinct: Count of distinct/unique values.

> **Filters:** The **Filters** option allows you to apply conditions to refine the displayed data. You can filter based on parameters such as:
>> **Date Range**: Select a specific time period to display data for that range.
**Category**: Filter by categories relevant to the metric, such as Product Lines, Regions, etc.
