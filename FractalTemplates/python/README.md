# Python transformation

# To create transformation you have to use an internal orphea package called funnel

This internal package is the decorator function where you can pass sources and target dataset.

The Python Transformation Program is a tool designed for data transformation, allowing users to modify and manipulate 
datasets to fit their desired output. To access this functionality, users must utilize an internal package known 
as "funnel," which serves as a decorator function for the program. Within Funnel, users can pass both source and 
target datasets to create the desired transformation. This tool provides a professional and efficient means of data 
manipulation for individuals seeking to optimize their data analysis processes.
