# Planification

<div style="text-align: justify"> Une planification est une tâche de génération récurrente qui assure un flux constant de données dans MoveToData. Le déclencheur dans une planification détermine quand la génération doit s'exécuter.

<p style="text-indent: 20px;">Une planification est considérée comme exécutée lorsque les conditions de déclenchement sont remplies et que la génération est effectuée. Si un programme est déclenché alors qu'un autre est encore en cours d'exécution, il restera en attente jusqu'à ce que le programme en cours soit terminé avant d'être exécuté.</p>

<p style="text-indent: 20px;">La planification et les builds sont liés. Vous pouvez définir des déclencheurs pour planifier les constructions d'ensembles de données dans MoveToData.</p>

## Création d'un horaire

<p style="text-indent: 20px;">Dans tous les ensembles de données, il existe une option de planification ci-dessous.</p>

![Schedule1](../../_media/schedule%20fr.png)

<p style="text-indent: 20px;">En sélectionnant l'option Planifier, vous pouvez choisir exactement quand vous souhaitez déclencher la génération de manière cohérente et automatique.</p>

### Personnalisation des déclencheurs

<p style="text-indent: 20px;">Vous pouvez personnaliser le déclencheur de planification jusqu'à la seconde en vous donnant toutes les options pour vous assurer que vous pouvez utiliser MoveToData à votre guise.</p>

![Schedule2](../../_media/schedule2%20fr.png)

<p style="text-indent: 20px;">Vous pouvez personnaliser soit par heure, soit par source.</p>

![Schedule3](../../_media/schedule3%20fr.png)

<p style="text-indent: 20px;">Pour vérifier votre tableau des horaires, cliquez sur les horaires dans le menu de la barre latérale.</p></div>
