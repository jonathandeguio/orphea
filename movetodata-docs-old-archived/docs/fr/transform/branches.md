# Branches

## Aperçu

<div style="text-align: justify"> Comme Git, MoveToData permet un système de contrôle de version distribué qui permet aux développeurs de travailler simultanément sur plusieurs versions d'une base de code. Les branches fournissent un moyen d'isoler le travail sur une fonctionnalité particulière ou un correctif de bogue sans affecter la base de code principale.

En général, les branches MoveToData permettent aux développeurs de travailler indépendamment sur différentes parties d'une base de code, de collaborer sur des fonctionnalités et des corrections de bogues, et de suivre les modifications apportées à la base de code au fil du temps, tout comme Git.

Voici une brève explication de la branche MoveToData :

<b><i><u>Branche principale</b></u></i>: La branche master est la branche principale du référentiel MoveToData. Il contient généralement la dernière version stable de la base de code.

## Comment créer une branche :

- Connectez-vous à votre compte
- Accédez à Projets à l'aide du menu de la barre latérale
- Sélectionnez votre projet actif
- Sélectionnez votre référentiel

## Comment fusionner une ou plusieurs branches

 <p style="text-indent: 20px;">Vous pouvez fusionner une ou plusieurs branches où les modifications apportées dans une branche sont ajoutées aux modifications apportées dans une autre branche. Cela se fait généralement lorsqu'un développeur souhaite incorporer les modifications qu'il a apportées dans une branche de fonctionnalités ou une branche de correction de bogues dans la branche principale (généralement la branche principale).</p>

<b><p style="text-indent: 20px;">Sélectionnez la branche que vous souhaitez fusionner avec la branche principale, et elle sera fusionnée dès que les modifications seront validées.</b></p>

![MergeBranch](../../_media/Transform/Branches/mergebranch%20fr.png)

<b><p style="text-indent: 20px;">Vous pouvez fusionner plusieurs branches dans la branche principale.</p>

<p style="text-indent: 20px;">Considérez la branche master comme la branche constante.</b></p>

## Comment supprimer des branches

<p style="text-indent: 20px;"> Une fois qu'un développeur a terminé de travailler sur différentes parties d'une base de code, en corrigeant des bogues, il peut souhaiter supprimer cette branche si elle ne lui sert pas à des fins futures.

- Pour supprimer une branche particulière, cliquez sur le menu déroulant de la branche et appuyez sur l'icône de corbeille pour supprimer la branche.

![DeleteBranch](../../_media/Transform/Branches/delete%20fr.png)

- MoveToData vous demandera de confirmer le nom de la succursale que vous souhaitez supprimer.</div>

![ConfirmDeletion](../../_media/Transform/Branches/delete1.1%20fr.png)
