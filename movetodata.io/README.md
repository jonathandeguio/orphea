# movetodata.io
This is static information site project

The 'Public' folder contains prod public content. A 'git push' will trigger an rsync to the storage bucket behind www.movetodata.io & movetodata.io.

The 'Public_Beta' folder contains beta/test content.  A 'git push' will trigger an rsync to beta.movetodata.io.

All other folders are ignored.

Setup wordpress using componse.yaml
