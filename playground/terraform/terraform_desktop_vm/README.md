
# Install basic web server in gcp using terraform

<br>

## Prereqs:  

- install terraform (see update_terrform.sh)
- setup gcloud sdk and set correct credentials (same as gcloud setup for GKE)
- export credentials for service account for use by terraform (see setup-creds.sh)

<br>

### To initialise / validate / see what's delivered / apply / destroy
```
terraform init
terraform validate
terraform plan
terraform apply
terraform destroy
```
