# Vagrant - Single CentOS 7 Machine with K8S Installed
## This node will have internet access!
<br>

Prerequisites:

https://www.vagrantup.com/downloads

https://www.virtualbox.org/

<br>
To Run/Stop/Destroy:

```
vagrant up

vagrant stop

vagrant destroy
```

<br>
Connect to the machine with:

```vagrant ssh```

<br>

For more info see:  
https://learn.hashicorp.com/collections/vagrant/getting-started


Official CentOS Vagrant/VirtualBox images: <br>
https://cloud.centos.org/centos/7/vagrant/x86_64/images/
