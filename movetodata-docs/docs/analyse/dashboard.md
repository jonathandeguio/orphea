# Dashboard

Dashboards are a way to display a range of charts and data right inside MoveToData. 
The dashboards implement a drag and drop solution to add and change different charts and views.

![dash1](../_media/docs_ss/analyze/dashboard.png)

## Features

When customising a dashboard, users can add all types of charts and even utilise headings and markdown to add additional information.
Within one dashboard, you can have multiple tabs using the top bar with options to change tab names for easy organisation. 

Users can access individual charts within dashboards and be redirected to them in order to edit them quickly without navigating to them manually. Clicking the 3 dots on any dashboard components opens up a range of options. Dashboards can be viewed in full screen by using the full screen option within the 3 dots menu.
