# Overview

## Integration

Data integration is one of the steps of data pre-processing before transformation that involves combining data residing in different sources and providing users with a unified view of these data.

MoveToData offers advanced data integration tools that go beyond traditional ETL or ELT solutions. Its design aims to lower the cost of data integration over time through a comprehensive set of capabilities that enhances the efficiency of data teams. While cloud services provide basic storage and compute for pipelines and experimentation, additional layers of functionality are necessary for managing, delivering, and validating datasets for critical operations. MoveToData serves as the data integration backbone for complex environments.

## Transformation

MoveToData's data transformation features include an extensible, scalable build system that utilizes multimodal compute to produce output datasets. Its compute-agnostic "Build" framework provides fully integrated security and data lineage, and enables mixing-and-matching of third-party compute runtimes.

It also includes an integrated suite of data transformation authoring, change management, data quality, pipeline scheduling, and metadata introspection capabilities that work together to provide a "mission control" for data engineers.

## Creating Pipeline

MoveToData creates pipelines with respect to the data subset it has connected and its pipeline management capabilities include change management, data quality, and data loading features. The Pipeline Builder application enables fast, flexible, and scalable delivery of data pipelines while providing robustness and security.

Data engineers can define a rigorous release process for production pipelines, including health checks that guarantee only fully compliant data will be deployed to production. Where issues are found, the platform provides diagnostics on the discrepancies detected. <!--Diagnostics are available in MoveToData's integrated analysis and modeling tools, as well as to any third-party tools accessing the outputs via REST APIs or other interfaces.-->
