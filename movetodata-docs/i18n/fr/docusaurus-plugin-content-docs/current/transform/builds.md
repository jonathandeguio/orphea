# Builds

Un build est le processus de compilation du code source dans une application logicielle ou une bibliothèque exécutable.

Le build implique généralement de compiler le code, de le lier à des bibliothèques externes et de créer un package installable pouvant être déployé dans un environnement cible. Il peut être automatisé à l'aide d'outils de construction comme Gradle. L'automatisation du build ainsi que la planification peuvent faire gagner du temps et assurer la cohérence entre les environnements.

Vous pouvez démarrer le build après avoir mis à jour un jeu de données particulier.

Vous cliquez simplement sur Build qui est l'icône du marteau ici et le processus commence automatiquement.

![Builds](../_media/Transform/transform/builds%20fr.png)

:::note
Il s'agit de la version normale.
:::

### Journal de build

Vous pouvez également afficher le journal de build qui fournit un enregistrement du build utilisé pour suivre la progression de la construction du logiciel. Un journal de build inclut généralement des informations sur le processus, telles que les commandes exécutées, les résultats de chaque commande et les erreurs ou avertissements qui se sont produits pendant le build.

![BuildLog](../_media/Transform/builds/buildlog%20fr.png)

Les journaux de build sont utiles pour un certain nombre de raisons. Ils fournissent un enregistrement détaillé du build, qui peut être utilisé à des fins de débogage si des erreurs se produisent. En analysant les journaux de build, les développeurs peuvent identifier les tendances et les modèles dans le processus de build, ce qui peut aider à identifier les opportunités d'amélioration.

### Historique de construction

Pour vérifier l'historique de vos builds, cliquez sur Builds à l'aide du menu de la barre latérale :

![BuildHistory](../_media/Transform/builds/buildhistory%20fr.png)

Dans l'historique de construction, vous pouvez afficher :

- L'identifiant de construction
- Le référentiel
- La branche
- La base de données
- Le type de déclencheur
- L'auteur
- L'horodatage
- La durée
- Le Spark
