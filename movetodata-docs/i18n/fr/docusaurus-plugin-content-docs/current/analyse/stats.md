# Statistiques

Les statistiques jouent un grand rôle dans l'utilisation optimale de MoveToData.
La plateforme vous permet d'afficher les statistiques de n'importe quelle colonne d'un ensemble de données.

## Accéder aux statistiques

![stat](../../../../../docs/_media/docs_ss/analyze/columnStats-usage.png)

## Sélectioner une colonne 

![stat](../../../../../docs/_media/docs_ss/analyze/dataset-columnStatsColumns.png)

Accédez à cette page via :

- Onglet Projets
- Sélectionnez un projet
- Sélectionnez les données
- Cliquez sur un jeu de données

Survolez une colonne et cliquez sur les trois lignes
L'option pour les statistiques de colonne devrait apparaître

![stat2](../../../../../docs/_media/docs_ss/analyze/columnsStats-result.png)

Ici, vous pouvez voir plusieurs types de statistiques concernant la colonne.
