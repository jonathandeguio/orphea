# Introduction to projects

A project is the central organizational unit in Orphea. It includes the Visualization (Charts, Dashboard), Logic (Data Pipeline), Data and Documentation. A project is isolated with security so that only a allowed set of users can access the space.

# Projects home

Your default landing page in Orphea is the Projects page. You can quickly find projects you created, as well as those shared with you and your workspace.

:::caution
You need to be Project Administrator to be able to create new projects. You can request a project administrator to create a new project for you.
:::

![Projects](../_media/docs_ss/Gettingstarted/Introductiontoprojects/ProjectsHome.png)

# Title & Description

The title and description of a project are visible in several places:

- In the project's Logic View
- In the Projects page
- Optionally, in the published app (see App metadata information)

:::tip
You can modify title and description simply by clicking and editing it.
:::
