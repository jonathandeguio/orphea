# Overview

## **What is Chart ?**
A chart is a visual representation of data designed to make complex information easier to understand, analyze, and interpret. Charts display data in a graphical format, using symbols such as bars, lines, slices, or points to represent values. They are widely used in various fields, including business, science, finance, and education, to communicate information clearly and concisely.

## **Various Charts on the Platform**
In our platform, we offer a diverse set of data visualization tools through the following chart types:

- **[Pie Chart](/analyse/charts/piechart.md):** A circular statistical graphic used to represent numerical proportions.
- **[Bar Chart](/analyse/charts/barchart.md):** A graphical representation that uses bars to show comparisons among categories or items.
- **[Line Chart](/analyse/charts/linechart.md):** A chart that displays information as a series of data points called 'markers' connected by straight line segments.
- **[Area Chart](/analyse/charts/areachart.md):** Similar to a line chart, but with the area below the line filled in, showing the volume of change.
- **[Scatterer Chart](/analyse/charts/scattererchart.md):** A chart that uses Cartesian coordinates to display values for typically two variables for a set of data.
- **[Horizontal Axis Chart](/analyse/charts/horizontalaxischart.md):** A chart where the data is displayed along a horizontal axis, often used for comparisons.
- **[Big Number Chart](/analyse/charts/bignumberchart.md):** A chart that highlights single values in a large format for emphasis.
- **[Table Chart](/analyse/charts/tablechart.md):** A chart that organizes data in rows and columns for easy comparison.
- **[Gauge Chart](/analyse/charts/gaugechart.md):** A circular gauge that represents a value within a specific range, often used for showing progress.
- **[Radar Chart](/analyse/charts/radarchart.md):** A chart that displays multivariate data in the form of a two-dimensional chart of three or more quantitative variables.
- **[Sunburst Chart](/analyse/charts/sunburstchart.md):** A multilevel pie chart used to visualize hierarchical data.
- **[Waterfall Chart](/analyse/charts/waterfallchart.md):** A chart that helps visualize the cumulative effect of sequentially introduced positive or negative values.
- **[Treemap Chart](/analyse/charts/treemapchart.md):** A chart that displays hierarchical data using nested rectangles.
- **[Word Cloud Chart](/analyse/charts/wordcloudchart.md):** A chart that displays words in varying sizes based on their frequency or importance.
- **[Map Chart](/analyse/charts/mapchart.md):** A chart that uses geographical maps to represent data points or statistical information associated with specific locations.
- **[Parameter Chart](/analyse/charts/parameterchart.md):** A chart that allows the visualization of data based on specific parameters.
