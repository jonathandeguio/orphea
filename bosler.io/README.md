# bosler.io
This is static information site project

The 'Public' folder contains prod public content. A 'git push' will trigger an rsync to the storage bucket behind www.bosler.io & bosler.io.

The 'Public_Beta' folder contains beta/test content.  A 'git push' will trigger an rsync to beta.bosler.io.

All other folders are ignored.

Setup wordpress using componse.yaml
